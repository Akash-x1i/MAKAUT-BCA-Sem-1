#include &lt;stdio.h&gt;
int main()
{
int a = 18, b = 3, c;

printf("a+b = %d \n", c);
c = a - b;
printf("a-b = %d \n", c);
c = a *b;
printf("a*b = %d \n", c);
c = a / b;
printf("a/b = %d \n", c);
c = a % b;
printf("Remainder = %d \n", c);
return 0;
}

ANSWER :-------------------------------------------

Output of the above program is :

2. If A=44 and B=12 then find A&B, A|B, A^B, ~A, ~B where &,
|, ^, ~ denotes bitwise AND, OR, Ex-OR and Complement
operator.

ANSWER----------------------------------------------

A=44
B=12

44 in BIN : 1011 00
12 in BIN : 0011 00

A&B  :  0011 00 = 12
A|B    :  1011 00 = 44
A^B   :  1000 00 = 32
~A     :   0100 11  = 19
~B     :   1100 11   = 51
